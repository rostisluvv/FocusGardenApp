# Focus Garden

MVP Android-приложения для ВКР «Разработка мобильного приложения на андроид для фокусирования внимания».

## Стек

- Kotlin
- Jetpack Compose
- Material 3
- MVVM
- Navigation Compose
- Room
- DataStore
- Coroutines / Flow

## Реализовано

- минималистичный главный экран;
- выбор длительности 10 / 25 / 45 / 60 минут через нижнее окно;
- выбор категории через нижнее окно;
- категории: Тело, Разум, Дело, Порядок, Люди;
- запуск фокус-сессии;
- обратный таймер;
- pixel-rain-анимация во время сессии;
- визуальный рост растения;
- успешное завершение сессии;
- ручное прерывание сессии;
- строгий режим: сворачивание приложения срывает активную сессию;
- сохранение результата в Room;
- создание растения GROWN / WITHERED;
- нижняя навигация из 3 иконок: Главная, Прогресс, Настройки;
- экран Прогресс: метрики, график 7 дней, Профиль баланса, Сад баланса, общий сад, история через bottom sheet;
- Профиль баланса в виде пятиугольника по категориям;
- Сад баланса с визуальным пределом LVL 0–5;
- настройки через DataStore;
- работа без сервера и интернета.

## Запуск

1. Открыть папку проекта в Android Studio.
2. Дождаться Gradle Sync.
3. Установить Android SDK 35, если Android Studio попросит.
4. Запустить app на эмуляторе или телефоне.

## Важно

Проект использует стабильную связку AGP 8.7.3 и Kotlin 2.0.21. Не обновляйте Android Gradle Plugin через Upgrade Assistant без необходимости.
