# Визуальные результаты тестирования

В этой версии приложения уже встроены демонстрационные данные для проверки экрана «Прогресс».

После первого запуска при пустой базе данных автоматически создаются:

- успешные фокус-сессии за последние 7 дней;
- несколько прерванных сессий;
- выращенные и засохшие растения;
- данные для графика «7 дней»;
- заполненный «Профиль баланса»;
- заполненный «Сад баланса» по категориям: Тело, Разум, Дело, Порядок, Люди.

Ожидаемый визуальный результат:

- «Профиль баланса» вытянут сильнее в сторону «Разум»;
- «Дело» развито средне;
- «Тело» развито слабее;
- «Порядок» и «Люди» развиты минимально;
- «Сад баланса» показывает уровни LVL 0–5;
- общий сад содержит выросшие и засохшие минималистичные растения;
- история содержит последние тестовые сессии.

Если приложение уже было установлено раньше, перед проверкой нужно очистить данные приложения или удалить приложение с эмулятора/телефона. Иначе демонстрационные данные не добавятся, потому что seeder работает только при пустой базе.

Быстрая проверка:

1. Запустить приложение.
2. Перейти в нижней панели на экран «Прогресс».
3. Проверить блоки:
   - метрики;
   - график за 7 дней;
   - Профиль баланса;
   - Сад баланса;
   - общий сад;
   - история через кнопку в правом верхнем углу.

## Обновление экрана «Прогресс»

- Общий блок «Сад» удалён, чтобы не перегружать экран.
- В блоке «Сад баланса» каждая категория стала входом в собственный сад.
- При нажатии на категорию открывается детальный сад зоны: Тело, Разум, Дело, Порядок или Люди.
- Внутри сада зоны отображаются растения только этой категории, включая выращенные и засохшие.
- Уровень зоны по-прежнему считается только по успешным минутам, чтобы прерванные сессии не усиливали баланс.
